## 基础篇

基础篇给大家推荐一些我看过和正在看的书：

* Linux 基本使用：《Linux运维之道第二版》
* Shell 脚本编程：《Linux Shell 核心编程指南》
* C 语言：《一站式学习 C 编程》 《C 语言非常道》 《嵌入式C语言自我修养》
* C++：《21天学通 C++ 》Siddhartha Rao
